# Criação de Máquinas Virtuais na Azure

A **SLA (Service Level Agreement)** define o nível de disponibilidade garantido pelo provedor.  
Por exemplo, uma SLA de **99%** permite um tempo de inatividade aproximado de **1,7 horas por semana**.  
Quanto maior o número de “noves” na SLA, maior a disponibilidade e menor será o tempo de indisponibilidade do serviço — essencial para demandas críticas de servidor.

## Passos para criar uma máquina virtual no Portal Microsoft Azure
- **Definir a região** onde a VM será hospedada
- **Escolher as opções de disponibilidade**
- **Selecionar o tipo de segurança**
- **Configurar as zonas de disponibilidade**
- Definir **outros parâmetros** (que serão abordados nas próximas aulas)
